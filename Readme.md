Taller 3: Maquetación Web con Bootstrap 🚀
​¡Bienvenido al repositorio del Taller 3! Este proyecto consiste en un sitio web multipágina completamente responsivo diseñado y estructurado utilizando el framework Bootstrap 5, aplicando buenas prácticas de maquetación visual adaptada a diferentes dispositivos.
​📂 Estructura del Proyecto
​El sitio web mantiene la siguiente organización de carpetas y archivos locales:  taller3/
├── app/
│   ├── forms.html       # Formularios de Registro e Inicio de Sesión
│   ├── tables.html      # Tabla de Gestión y Consulta de Clientes
│   └── github.html      # Guía e Información de comandos Git/GitHub
├── assets/
│   ├── css/             # Estilos personalizados (Bootstrap local)
│   ├── img/             # Recursos visuales (Logos, imágenes del carrusel)
│   └── js/              # Archivos de lógica JavaScript
├── index.html           # Página Principal (Inicio)
└── README.md            # Documentación del proyecto

🛠️ Tecnologías Utilizadas
​HTML5: Estructuración semántica del contenido global.  
​Bootstrap v5.3.3: Framework CSS nativo para un diseño ágil y adaptable sin sobrecargar código.  
​Bootstrap Icons v1.13.1: Librería de iconos incorporada en barras de navegación, botones y secciones informativas.  
​🖥️ Características de las Páginas
​1. Página de Inicio (index.html)
​Navegación: Barra superior fija (sticky-top) oscura con menú desplegable e integración de iconos.  
​Componente Carrusel: Deslizador de imágenes adaptivo (carousel slide) con indicadores de navegación para destacar multimedia.  
​Secciones Informativas: Distribución en rejilla (row, col-sm-3) con tarjetas de servicios, notas importantes y llamados a la acción estructurados.  
​2. Formulación y Acceso (app/forms.html)
​División de pantalla balanceada en dos columnas para Registro de Usuarios e Iniciar sesión.  
​Uso de controles de formulario nativos de Bootstrap (form-control, form-check) adaptados para dispositivos móviles.  
​3. Consulta de Clientes (app/tables.html)
​Diseño optimizado para datos masivos mediante tablas responsivas (table-responsive, table-sm).  
​Visualización organizada de un listado detallado de 20 clientes corporativos simulados.  
​4. Documentación Git (app/github.html)
​Espacio académico diseñado como una guía interactiva con el paso a paso de configuración e inicialización de repositorios.
