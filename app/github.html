Guia Github ADSO

1. Estructura

El proyecto tiene una estructura organizada para un sitio web de varias paginas: 

taller3/
├── app/
│   ├── forms.html
│   ├── tables.html
│   └── github.html
│
├── assets/
│   ├── css/
│   ├── img/
│   └── js/
│
├── index.html
└── README.md
Elemento Descripción
app/forms.html Página destinada a formularios
app/tables.html Página destinada a tablas
app/github.html Página destinada a la información guía para el manejo en GitHub.
app/assets/css Carpeta para archivos de estilos CSS, si el proyecto los requiere.
app/assets/img Carpeta para imágenes del proyecto
app/assets/js Carpeta para archivos JavaScript.
README.md Archivo README con documentación.
index.html Página principal del proyecto.
nota: esta guía es solo bootstrap para diseño Visual

2. Requisitos iniciales

  • Tener una cuenta activa en GitHub.
  • Tener instalado Git en el computador.
  • Tener instalado Visual Studio Code.

3. Configuración de Git desde Visual Studio Code

Abre la terminal integrada de Visual Studio Code:

Menú: Terminal → New Terminal

Configura tu nombre y correo a nivel global de forma obligatoria:

git config --global user.name "Tu Nombre"
git config --global user.email "tu_correo@email.com"

4. Inicializar Git en el proyecto

Ubícate en la carpeta raíz del proyecto, donde está el archivo index.html, y ejecuta:

git init
git add .
git commit -m "Primer commit"

Nota: Este primer commit guarda la versión inicial del proyecto, incluyendo index.html, forms.html, tables.html y assets.

5. Crear repositorio en GitHub

  1. Ingresar a GitHub.
  2. Hacer clic en el botón New repository.
  3. Asignar un nombre al repositorio, por ejemplo: proyecto-bootstrap-adso.
  4. Seleccionar si el repositorio será público o privado.
  5. Hacer clic en el botón inferior Create repository sin agregar archivo README si ya tienes código local.

6. Conectar proyecto local con GitHub

Copia la URL segura del repositorio creado en tu plataforma de GitHub y ejecuta:

git remote add origin https://github.com/usuario/proyecto-bootstrap-adso.git
git branch -M main
git push -u origin main

Importante: Recuerda cambiar usuario y proyecto-bootstrap-adso por tus datos reales de la cuenta de GitHub.
